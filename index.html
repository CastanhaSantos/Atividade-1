Este site tem como objetivo o aprendizado!

Aqui vamos aprender a utilizar as ferramentas Git e Github.

Git: É um software de versionameto e auxilia bastante quando se tem uam equipe trabalhando, e varias versões do mesmo projeto são geradas, o Git é oque chamamos de repositório local.

Github: Diferente do Git o Github é um repositório remoto onde você irá disponibilizar o seu código de maneira publica ou privada para que outros integrantes da equipe ou até mesmo a comunidade tenham acesso ao código.
